App Sharing Feature
How App Sharing Works

As an admin, you have access to all apps in the system. Regular users can only see apps that have been shared with them.

Admin View

As an admin, you can:

  • View all apps in the system
  • Share apps with specific users
  • Manage user access on app detail pages
User View

Regular users can:

  • View only apps shared with them
  • Download and install shared apps
  • See app details and version history
Sharing an App

To share an app with users, go to the app's detail page and use the "Manage Sharing" button, or choose a user from the dropdown in the "Shared With" section.
