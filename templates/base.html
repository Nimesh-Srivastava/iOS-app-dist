App Sharing Feature
How App Sharing Works

Admins have access to all apps in the system. Developers and testers can only see apps that they own or that have been shared with them.

Admin View

As an admin, you can:

  • View all apps in the system
  • Share apps with specific users
  • Manage user access on app detail pages
Developer View

Developers can:

  • View all apps in the system
  • Upload and manage apps
  • Build apps from GitHub
  • Share apps with developers and testers
Sharing an App

To share an app with users, go to the app's detail page and use the "Manage Sharing" button, or choose a user from the dropdown in the "Shared With" section.
